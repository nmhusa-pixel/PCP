# Pain Referral Assistant

This is a browser-based prototype for primary care pain management referrals.

Open `index.html` in a browser. The app screens for urgent red flags, checks whether conservative care and workup are documented, scores referral readiness, and generates a referral note.

Clinical sources embedded in the app:

- VMFH Back Pain Referral Algorithm for Primary Care
- American Academy of Pain Medicine primary care referral recommendations
- CDC Clinical Practice Guideline for Prescribing Opioids for Pain, 2022
- ACR Appropriateness Criteria for Low Back Pain
- ACP noninvasive low back pain guideline

This is clinical decision support only and needs clinician validation before operational use.
